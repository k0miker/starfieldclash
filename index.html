Game Instructions

Control the spaceship with your mouse or touch.

Avoid comets, collect red special stars for bonus points.

Use the following keys to adjust the game:

  • Arrow Up: Increase speed
  • Arrow Down: Decrease speed
  • W: Increase speed to 5x current speed while held
  • S: Set speed to current speed / 2 while pressed
  • NumPad +: Increase number of stars
  • NumPad -: Decrease number of stars
